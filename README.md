# TabaxiTable 2.1.5

Локальный онлайн-стол для классической D&D 5e 2014: комнаты по коду, синхронные листы персонажей, физические 3D-кубики и ручная VTT-карта. TabaxiTable помогает вести игру, но не играет вместо партии.

Текущие схемы данных:

- лист персонажа — **11**;
- сцена — **10**.

Старые комнаты, персонажи и сцены обновляются через нормализацию при загрузке.

## Запуск на Windows

1. Установи Node.js 20 или новее.
2. Запусти `start.bat`.
3. При ручном запуске используй:

```powershell
npm.cmd install
npm.cmd start
```

По умолчанию стол открывается на `http://localhost:31777`.

Постоянные данные хранятся отдельно от кода:

```text
data/rooms.json   комнаты и листы
data/assets/      загруженные карты и токены
data/backups/     автоматические копии комнат
```

### Быстрый лист на карте

Игрок сам выбирает доступные страницы и закрепляет любимые:

- навыки;
- спасброски;
- атаки;
- заклинания.

В панели доступны обзор, бой, проверки, подготовленная магия, цели и заметки. Формулы запускают физические публичные или закрытые броски без выхода с карты.

## Инструменты ведущего

- массовые HP, инициатива, размер, поворот, прозрачность, зрение, цвет, видимость и блокировка;
- выбор всех персонажей, NPC или токенов сцены;
- сохранение выделенных NPC как шаблона встречи;
- копирование выделения между сценами;
- диагностика размера комнаты и ресурсов;
- выбор качества и материала кубиков;
- перезапуск физического слоя;
- восстановление последней автоматической копии.

Автокопия создаётся сервером не чаще одного раза в пять минут при изменениях комнаты.

## Горячие клавиши карты

Горячие клавиши работают по физическим клавишам при русской и английской раскладке:

- `V` — выбор;
- `H` — рука;
- `M` — линейка;
- `P` — указатель;
- `L` — линия;
- `R` — прямоугольник;
- `C` — круг;
- `N` — конус;
- `D` — карандаш;
- `T` — текст;
- `K` — кубики;
- `F` — скрыть область туманом;
- `Shift + F` — открыть область;
- `Shift + S` — быстрый лист;
- `Пробел + ЛКМ` или средняя кнопка — панорамирование;
- `Ctrl + C/V/D`, `Ctrl + Z/Y` — копирование, дублирование, отмена и повтор.

При вводе текста и чисел хоткеи не срабатывают.

## NPC-листы 2.1.1

Ведущий может открыть настройки любого NPC-токена и создать для него компактный лист: характеристики, спасброски, проверки, атаки, урон и произвольные формулы. Для каждого значения или действия отдельно задаётся публичность. Публичные данные видны игрокам в быстром листе NPC, приватные физически удаляются из отправляемого им состояния комнаты.

Формулы NPC бросаются физическими кубиками на карте. Публичная строка создаёт общий бросок, приватная — закрытый бросок ведущего.
