# TabaxiTable

Онлайн-стол для классической D&D 5e: комнаты по коду, синхронные интерактивные листы персонажей и общие броски костей.

## Запуск на Windows

1. Установи [Node.js](https://nodejs.org/) версии 20 или новее.
2. Открой папку проекта в Проводнике.
3. Кликни по адресной строке папки, введи `cmd` и нажми Enter.
4. Выполни `npm install`.
5. Выполни `npm start`.
6. Открой `http://localhost:31777`. После первого входа TabaxiTable сам вернёт тебя в комнату при обновлении страницы.

## Обновление одной кнопкой

Если проект был один раз получен через GitHub (`git clone` или GitHub Desktop), запускай `update.bat`. Он подтянет только изменившиеся файлы и обновит зависимости. Комнаты из `data/rooms.json` и локальные настройки не отправляются в GitHub и не затираются обновлением.

Для игры через интернет проект нужно разместить на сервере. Пока сайт запущен только на компьютере, зайти на него сможет лишь хозяин компьютера и, при дополнительной настройке сети, люди из той же локальной сети.

## Уже работает

- создание комнаты и вход по шестизначному коду;
- роли ведущего и игрока;
- отдельный лист каждого персонажа;
- автоматические модификаторы характеристик, спасбросков и навыков;
- характеристики, HP, КД, атаки, снаряжение, заклинания и заметки;
- сложные формулы бросков, включая несколько типов костей;
- структурированные атаки с отдельными бросками попадания и урона;
- урон, лечение, временные HP, кости хитов и продолжительный отдых;
- состояния, истощение, монеты и расходуемые ячейки заклинаний;
- экспертиза навыков и броски навыков/спасбросков нажатием на бонус;
- предметы с количеством, весом, экипировкой, настройкой и магическими свойствами;
- автоматический вес инвентаря и грузоподъёмность;
- ресурсы и заряды с восстановлением после короткого или долгого отдыха;
- гримуар, подготовленные заклинания, концентрация, ритуалы и сотворение повышенной ячейкой;
- встроенный справочник заклинаний с поиском по названию, уровню и классу;
- компактные вкладки листа и броски без ухода на отдельную страницу;
- восстановление комнаты и несохранённого черновика после обновления страницы;
- цели, отдельные заметки, биография, внешность, владения и языки;
- экспорт и импорт персонажа в JSON;
- просмотр листов других участников;
- общие броски костей в реальном времени;
- сохранение комнат в `data/rooms.json`.

## Что важно знать

- Просто запускай `start.bat`: он сам перейдёт в папку проекта, установит зависимости при первом запуске и откроет браузер.
- Не запускай второй экземпляр одновременно. Если порт занят, TabaxiTable попробует следующий свободный порт и напишет точный адрес в окне.
- Кнопка с логотипом выходит из комнаты и забывает автовход. Обычное обновление страницы, наоборот, возвращает в текущую комнату.
- Справочник — стартовый набор из 55 базовых заклинаний с краткими описаниями; хоумбрю по-прежнему можно добавить вручную.
