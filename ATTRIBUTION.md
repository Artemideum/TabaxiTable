# Атрибуция открытых правил

Каталог предметов и часть правил TabaxiTable используют материал **System Reference Document 5.1** компании Wizards of the Coast LLC.

Материал SRD 5.1 доступен по лицензии **Creative Commons Attribution 4.0 International (CC BY 4.0)**:

- источник: https://dnd.wizards.com/resources/systems-reference-document
- лицензия: https://creativecommons.org/licenses/by/4.0/

Русские названия, короткие интерфейсные пояснения, структура данных и программная автоматизация подготовлены для TabaxiTable. Тексты закрытых книг дословно не воспроизводятся.

# Сторонние библиотеки

Физические 3D-кубики используют **@3d-dice/dice-box-threejs** — Three.js/Cannon ES dice roller, распространяемый по лицензии MIT. Копия лицензии находится в `public/vendor/dice-box-threejs.LICENSE.txt`.
